# CityLunch

Учебный проект по веб‑разработке для лабораторных работ студента Денисенко Даниила, группа 241‑326.

Здесь реализован адаптивный лендинг службы доставки бизнес‑ланчей **CityLunch**: главная страница, конструктор ланча и базовая верстка меню с использованием HTML и CSS.

Проект предназначен для отработки навыков вёрстки, работы с макетами и оформления кода под GitHub.
